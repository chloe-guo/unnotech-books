## 你對於所有使用到的第三方 library 的理解，以及為何使用它
> axios
* 優點：設計上簡潔，API 串接使用上簡單，支持瀏覽器和node.js，能很好的與各種前端框架整合，且支持 Promise API。
* 缺點：不支持較古老的瀏覽器。

> tailwind css
* tailwind css 是一 css 框架是利用 class name 為元素加上 css 樣式；除了他本身的 css 樣式外，還能自行客製化新增樣式，也支援 RWD 。
